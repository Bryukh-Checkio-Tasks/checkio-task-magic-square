In mathematics, a magic square is an arrangement of numbers (integers in our case) in a square grid, where the numbers in each row, and in each column, and the numbers in the forward and backward main diagonals, all add up to the same number.

Given not completed magic square (with size from 3 to 5). You should finish it. It must be a normal magic square and contains integers from 1 to n2 without repeating. The square is presented as a list of list with integers. Zero is a empty cell. You should return completed magic square. The task can have several solutions.

You can read about Magic square at Wikipedia.

Input: Partially filled a magic square. A list of list with integers.

Output: Completed magic square. A list of list with integers.

Example: 

checkio([
    [2, 7, 6],
    [9, 5, 1],
    [4, 3, 0]
])
#must return [[2, 7, 6], [9, 5, 1], [4, 3, 8]]

checkio([
    [0, 0, 0],
    [0, 5, 0],
    [0, 0, 0]
])
#can return [[2, 7, 6], [9, 5, 1], [4, 3, 8]] or
# [[4, 9, 2], [3, 5, 7], [8, 1, 6]

checkio([[1, 15, 14, 4],
         [12, 0, 0, 9],
         [8, 0, 0, 5],
         [13, 3, 2, 16]])
# answer [[1, 15, 14, 4], [12, 6, 7, 9], [8, 10, 11, 5], [13, 3, 2, 16]]
